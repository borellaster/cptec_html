Formatação dos arquivos CSV utilizando a ferramenta R.

Primeiramente deve-se instalar o software R acessando o link: https://www.r-project.org

Posteriormente instalar o Rstudio no link:https://www.rstudio.com/

O arquivo em R pode ser baixado no link: converte_csv.R

Abrir o arquivo R utilizando o Rstudio e localizar a seguinte linha de comando:

setwd("C:/Users/Usuário/Desktop")

Nela deverá ser colocado o caminho onde está o arquivo CSV original.

Para rodar no Windows para digitar "Rscript converte_csv.R nome_do_arquivo_sem_a_extensão_csv"

Exemplo: Rscript converte_csv.R Requisicao_558

No Linux deve-se deixar o script como executável e digitar "./converte_csv.R nome_do_arquivo_sem_a_extensão_csv"

Exemplo: Rscript converte_csv.R Requisicao_558
